# wiklet
A local desktop notes tool built aimed at worldbuilding - characters, factions, locations and anything more.
--
This is a personal project built around my own workflow and specific features I actually would use.
## Running the project
```bash
git clone https://github.com/Valkirpod/wiklet
cd wiklet
pip install -r requirements.txt
python main.py
```
You can try cloning develop branch to get the most up-to-date project, but it might be unstable.
## Planned features
- Collections of entries with picture card grid view
- Per-entry text content with basic markdown
- Relationship graph view between entries
- Custom metadata fields per collection (text, numbers, dates, selects)
- Sidebar with categorized collections
## Built with
- Python
- PySide6
